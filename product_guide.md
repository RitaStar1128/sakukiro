# サクキロ (SAKUKIRO) プロダクトガイド

![SAKUKIRO OGP](/images/ogp-generated.png)

## 1. プロダクト概要

**サクキロ (SAKUKIRO)** は、「記録の速さ」を極限まで追求した支出管理・家計簿アプリケーション（PWA）です。

「家計簿が続かない最大の理由は、入力の手間にある」という仮説に基づき、起動から入力完了までのステップを最小化しました。複雑な分析機能やクラウド同期をあえて排除し、ユーザーが「財布から小銭を出すよりも速く」支出を記録できる体験を提供します。

## 2. 主要機能

サクキロは「削ぎ落とすこと」で価値を生み出しています。

### 機能詳細
*   **インスタント・キーパッド**: アプリ起動と同時にテンキーがアクティブになり、即座に金額入力が可能です。ロード時間やログイン画面による待機時間はゼロです。
*   **サムゾーン・インターフェース**: スマートフォンでの片手操作を前提に、入力キーと確定ボタンを画面下部（親指の可動域）に集約しています。
*   **マルチカレンシー対応**: 日本円 (JPY) だけでなく、米ドル (USD)、ユーロ (EUR)、人民元 (CNY) に対応。設定画面から瞬時に切り替えられ、海外旅行や出張時の支出管理にも最適です。
*   **CSVエクスポート**: 蓄積されたデータは標準的なCSV形式で出力可能。ExcelやGoogleスプレッドシートなどの外部ツールで、詳細な集計やグラフ化を行いたいユーザーのニーズに応えます。

### 他ツールとの違い
一般的な家計簿アプリが「多機能・自動連携」を目指すのに対し、サクキロは「単機能・手動入力の高速化」を目指しています。
銀行口座連携やレシート撮影機能はありませんが、それらの機能が持つ「反映のタイムラグ」や「読み取り精度のストレス」とは無縁です。

## 3. よくある質問 (FAQ)

**Q. アプリのインストールは必要ですか？**
**A.** App StoreやGoogle Playからのインストールは不要です。ブラウザでアクセスし、「ホーム画面に追加」を行うことで、ネイティブアプリと同様に使用できます。

**Q. 機種変更をした場合、データは引き継げますか？**
**A.** 現行バージョンでは、データはブラウザ内に保存されるため、自動的な引き継ぎ機能はありません。機種変更前にCSVエクスポート機能を使ってデータをバックアップし、新しい端末で参照・管理することをお勧めします。

**Q. オフラインでも使えますか？**
**A.** はい、PWA (Progressive Web App) 技術を採用しているため、電波の届かない場所や機内モードでも問題なく起動・記録が可能です。

**Q. 入力したデータは誰かに見られますか？**
**A.** いいえ。データはお客様の端末内（ブラウザのローカルストレージ）にのみ保存され、外部のサーバーへ送信されることは一切ありません。プライバシーは完全に守られます。

## 6. 始め方ガイド

### ステップ 1: アクセス
スマートフォン（iPhone/Android）のブラウザで、サクキロのURLにアクセスします。

### ステップ 2: ホーム画面に追加
*   **iPhone (Safari)**: 画面下部の「共有」アイコン → 「ホーム画面に追加」をタップ。
*   **Android (Chrome)**: 画面右上のメニューアイコン → 「アプリをインストール」または「ホーム画面に追加」をタップ。

### ステップ 3: 初期設定（任意）
アプリを起動し、右上の設定アイコン（歯車）から以下を設定できます。
*   **言語**: 日本語 / English
*   **通貨**: JPY / USD / EUR / CNY
*   **テーマ**: ライト / ダーク / システム準拠

これだけで準備は完了です。さあ、最初の支出を記録してみましょう。

## 7. 技術仕様

サクキロは、最新のWeb標準技術を用いて構築されています。

### システム要件
*   **OS**: iOS 14.0以上, Android 8.0以上, Windows/macOS (最新版推奨)
*   **ブラウザ**: Google Chrome, Safari, Firefox, Edge の各最新版
*   **JavaScript**: 有効化必須

### 技術スタック
*   **Frontend**: React 19, TypeScript, Vite
*   **Styling**: Tailwind CSS v4 (Neo-Brutalism Design System)
*   **Storage**: LocalStorage API (No Backend Database)
*   **PWA**: Service Worker (Cache-first strategy), Web App Manifest

### API仕様
現行バージョン（v1.0）はクライアントサイドのみで完結するアーキテクチャを採用しているため、外部公開APIは提供していません。
データ連携が必要な場合は、CSVエクスポート機能をご利用ください。CSVフォーマットは以下の通りです。

```csv
date,amount,category,note,currency
2024-01-20T10:00:00.000Z,1500,cat_food,ランチ,JPY
2024-01-20T18:30:00.000Z,320,cat_transport,電車代,JPY
```
